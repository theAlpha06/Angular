# <span style="color:yellow">TypeScript</span>

- A strongly typed programming language
- Created and maintained by Microsoft
- Superset of JavaScript

## <span style="color:yellow">Introduction to TypeScript</span>

- Compiles down to JS
- Keeps your code evergreen
- Supported by all major libraries and frameworks

## <span style="color:yellow">Type Safety</span>

- Keeps your application free from Type Errors
- Languages like C#, Java are examples of Type Safe Language
- Keeps your JS code free from undefined and null values
- In TypeScript Types are stripped when your code is converted to JS
- Deno: New runtime env which understands TypeScript, unlike browsers.
